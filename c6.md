# ESP32-C6 Smart Weather & Clock Display (Lightweight Build)

A sleek, ultra-low-overhead ESPHome firmware configuration (`c6.yaml`) for the **ESP32-C6 DevKitC-1** featuring a 128x64 OLED display (`SH1106_128X64`). This lightweight build focuses on energy efficiency, minimal CPU memory footprint, high-contrast typography, and active BLE device tracking.

---

## Key Features

- 🌤️ **Clean Telemetry Layout**: High-contrast display featuring indoor and outdoor temperatures with real-time trend indicators (`^`, `v`, `=`) alongside a sleek clock (`14:32`).
- 🔤 **Preserved Modern Typography**: Uses high-grade typography (`Fira Code` and `monacottf.otf`) matching the exact pixel alignment and style definitions of the original design system.
- 📡 **Active BLE Device Tracking & Beaconing**: Integrated `esp32_ble_tracker` for Home Assistant room-presence tracking of BLE tags/devices and `esp32_ble_beacon` for iBeacon broadcasting.
- ⚡ **Minimalist CPU & Power Footprint**: Operates without heavy particle animation rendering loops or high-frequency canvas updates, drastically reducing RAM and power consumption.

## Hardware Specifications

| Hardware Component | Details |
| :--- | :--- |
| **Microcontroller** | Espressif ESP32-C6 DevKitC-1 (RISC-V, 4MB Flash, Wi-Fi 6, BLE 5) |
| **Framework** | `esp-idf` |
| **Display** | SH1106 / SSD1306 128x64 Monochrome I2C OLED |
| **I2C Pins** | SCL: GPIO 7, SDA: GPIO 6 (`400kHz` frequency) |

---

## Target Build Files

- **`c6.yaml`**: Primary ESPHome target for ESP32-C6 device.
- **`c6_sim.yaml`**: Desktop SDL host simulator target for testing the lightweight UI on Linux.
- **`packages/c6/c6_display_lightweight.yaml`**: Low-overhead display package.
- **`packages/c6/c6_ble.yaml`**: BLE device tracking & iBeacon package.
