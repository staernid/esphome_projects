# ESP32-C6 Smart Weather & Clock Display

An advanced, high-performance ESPHome firmware configuration for the **ESP32-C6 DevKitC-1** featuring a 128x64 OLED display (`SH1106_128X64`) powered by LVGL version 9 and a **hardware-accelerated algorithmic particle/vector rendering engine** showcasing the 160MHz RISC-V compute performance of the ESP32-C6.

---

## Showcase Features

- **Debug Mode Animation Showcase**: Enabled by default (`debug_mode: true`). Automatically cycles through all 8 algorithmic weather animations every 5 seconds so you can preview and verify all visual effects without waiting for weather changes!
- **Home Assistant Data Ingestion**: Real-time integration with Home Assistant for indoor temperature sensors and weather forecast entity data (`weather.forecast_home`).
- **Dynamic Dual-Page Interface**:
  - **Page 1 (Indoor & Time)**: Top-left indoor temperature statistics and bottom-right digital clock.
  - **Page 2 (Outdoor Forecast)**: Large outdoor temperature readout and forecast condition text.
  - **Auto-Page Rotation**: Smooth page switching loop every 6 seconds.
- **Hardware-Accelerated 60 FPS Engine**: Overclocked I2C bus (`800kHz`) and synchronized hardware display refresh (`update_interval: 16ms`) for ultra-fluid vector animations.
- **Algorithmic Weather FX Showcase (Overlay Canvas)**: Zero UI widget bloat. All weather visuals are procedurally rendered in C++ vector math and a 16-particle hardware simulation engine onto a single transparent LVGL overlay canvas (`top_layer`):
  - ☀️ **Sunny / Clear Day**: Core sun circle with 4 orbiting rotating ray flares and atmospheric heat shimmer motes floating upward.
  - 🌙 **Clear Night**: Sharp geometric crescent moon with a 16-point phase-shifted twinkling starfield and procedural meteor/shooting star streaks.
  - ☁️ **Cloudy / Partly Cloudy**: Dual-layer volumetric cumulus clouds with parallax drifting (foreground fast cloud drifting over background slow cloud).
  - 🌧️ **Torrential Rain / Pouring**: 16 dynamic rain streaks falling at high velocity ($vy \ge 4.6\text{px/step}$) with wind slant.
  - ❄️ **Snowy Blizzard**: 16 multi-sized snowflakes ($1\times1, 2\times2, 3\times3$) fluttering with dual-axis trigonometric sine turbulence.
  - ⚡ **Thunderstorm**: Real-time procedural jagged electric lightning bolts flashing across dark storm clouds during thunder strikes.
  - 💨 **Windy / Fog**: Fast vector stream lines sweeping rightwards with swirling dust/leaf particles executing orbital sine-wave drifts.

## Hardware Specifications

| Hardware Component | Details |
| :--- | :--- |
| **Microcontroller** | Espressif ESP32-C6 DevKitC-1 (RISC-V, 4MB Flash, Wi-Fi 6, BLE 5, Thread/Zigbee) |
| **Framework** | `esp-idf` |
| **Display** | SH1106 / SSD1306 128x64 Monochrome I2C OLED |
| **I2C Pins** | SCL: GPIO 7, SDA: GPIO 6 (`800kHz` frequency) |

### Key Configuration Highlights in `c6.yaml`

#### 1. Hardware & Display Setup
```yaml
i2c:
  - scl: 7
    sda: 6
    id: i2c_1
    frequency: 800khz

display:
  - platform: ssd1306_i2c
    i2c_id: i2c_1
    model: SH1106_128X64
    id: display_ssd1306_i2c_1
    auto_clear_enabled: false
    update_interval: 16ms # Unlocks 60 FPS physical screen flushing
```

#### 2. LVGL Overlay Canvas & Pages
```yaml
lvgl:
  id: my_lvgl
  displays: display_ssd1306_i2c_1
  update_interval: 16ms
  top_layer:
    widgets:
      - canvas:
          id: weather_canvas
          width: 128
          height: 64
          transparent: true
```

#### 3. Algorithmic Renderer Engine (`interval: 33ms`)
Every 33ms (~30 FPS), a C++ animation lambda advances particle physics, clear transparent canvas buffers, and evaluates dynamic trigonometric formulas for active weather effects. All screen coordinates are strictly bounded to $[0, 127] \times [0, 63]$ for memory-safe LVGL 9 rendering.
